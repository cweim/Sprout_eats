<task type="auto">
  <name>Task 1: Extend search to address, notes, types</name>
  <files>webapp/app.js</files>
  <action>
Update applyFilters() search logic:

Current (approximate):
```javascript
if (searchQuery && !place.name.toLowerCase().includes(searchQuery.toLowerCase())) {
    return false;
}
```

New:
```javascript
if (searchQuery) {
    const q = searchQuery.toLowerCase();
    const searchFields = [
        place.name,
        place.formatted_address,
        place.notes,
        place.place_types
    ].filter(Boolean).map(s => s.toLowerCase());

    if (!searchFields.some(field => field.includes(q))) {
        return false;
    }
}
```

This searches name, address, notes, and types (comma-separated string).
