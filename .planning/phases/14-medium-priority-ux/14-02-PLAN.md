<task type="auto">
  <name>Task 2: Detect Telegram theme and apply</name>
  <files>webapp/app.js</files>
  <action>
Add theme detection on app initialization:

1. Create applyTheme() function:
   - Check Telegram.WebApp.colorScheme ('light' or 'dark')
   - Set document.documentElement.dataset.theme = colorScheme
   - If not in Telegram (colorScheme undefined), check prefers-color-scheme

2. Call applyTheme() early in initializeApp()

3. Listen for theme changes (Telegram can switch mid-session):
   - Telegram.WebApp.onEvent('themeChanged', applyTheme)

4. Handle edge cases:
   - If Telegram SDK not loaded, use media query fallback
   - Log theme detection for debugging
  </action>
  <verify>Open Mini App with Telegram dark mode → app shows dark theme</verify>
  <done>Theme detection works, respects Telegram colorScheme</done>
</task>
