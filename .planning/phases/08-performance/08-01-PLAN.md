1. Add `preload_model()` function that loads the model eagerly:

```python
def preload_model():
    """Pre-load Whisper model into memory. Call at startup."""
    global _model
    if _model is None:
        _model = whisper.load_model(config.WHISPER_MODEL)
    return _model
```

2. Add `is_model_ready()` function to check status:
```python
def is_model_ready() -> bool:
    """Check if Whisper model is loaded and ready."""
```
